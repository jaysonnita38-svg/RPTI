## 🌟 项目简介

最近 SBTI人格测试火遍全网，但说唱圈的人格测试大多还停留在简陋的 P 图阶段。为了给大家提供一个真正能跑、能算、能“钓鱼”的娱乐工具，**RPTI (Rap Personality Type Indicator)** 应运而生。

这是一个基于 **React + Tailwind CSS** 构建的纯前端单文件测试应用。它不问你喜欢什么 Flow，也不问你听什么伴奏，而是通过 **15 道极度抽象、甚至带点地狱笑话色彩的生活场景题**，剥开你的伪装，直击你灵魂深处的地下基因。

## 🔥 核心特色

### 1. 阵容空前：30 位灵魂人格
项目内置了 30 位中国说唱圈最炙手可热（且极具梗度）的歌手。从主流说唱歌手GAI、ASEN，到抽象界的“成都迪士尼”诺米，再到“加拿大少爷”严浩翔。每个人格都配有深度挖掘的“扎心”评语，拒绝套路，字字珠玑。

### 2. 交互进化：拒绝盲目跳转
区别于传统的自动跳题逻辑，RPTI 允许答题者在当前页面反复横跳。
- **自主选择**：选中后不立即跳转，支持反复修改。
- **全自由回溯**：页面左下角设置“上一题”按钮，选错了？想改了？随时回去。

### 3. 顶级彩蛋：强制触发机制
项目内置了一个检测雷达，一旦答题者选择了那个神秘的选项，系统将瞬间锁死底层逻辑——无论后续 14 道题选得多么 Hardcore，最终结果都将强制导向彩蛋人物。

### 4. 极致视觉：多巴胺审美
采用高饱和度的多巴胺渐变背景搭配时下流行的 **毛玻璃（Glassmorphism）** UI 效果。不仅好玩，发到朋友圈更是颜值担当。

## 🛠️ 技术实现

- **React (CDN 版)**：无需复杂安装，单文件即可运行。
- **Tailwind CSS**：提供丝滑的响应式布局和动画。
- **Babel Standalone**：在浏览器端实时编译 JSX。
- **Web Share API**：支持唤起手机原生分享面板，一键发送给兄弟们。

## 🚀 部署指南

本项目采用“全代码封装于单一 HTML”的方案，部署极其简单：

1. **GitHub Pages**：新建仓库，将代码重命名为 `index.html` 上传，开启 Pages 即可。
2. **Netlify Drop**：将包含 `index.html` 的文件夹直接拖拽至部署区域，30 秒生成永久链接。
